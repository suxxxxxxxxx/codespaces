# 十进制转二进制

# 所有的位运算方法都是为二进制服务的

-按位与(&)
将两个数字的二进制的每一位进行 & 操作，如果两个相应位置都为1，则结果为1，否则结果为0

-按位或(|)
将两个数字的二进制的每一位进行 | 操作，如果两个相应位置至少一个为1，则结果为1，否则结果为0

-按位异或(^)
将两个数字的二进制的每一位进行 ^ 操作，如果两个相应位置不同，则结果为1，否则结果为0

-按位非(~)
将两个数字的二进制的每一位进行取反操作，0变成1，1变成0，将得到的值转成32位的有符号整型
